# Setup

Install git-lfs to pull files automatically
```
sudo apt-get update
sudo apt-get install git-lfs
```
or
```
brew install git-lfs
```
then
```
git lfs install
```
now, you can clone
```
git clone git@github.com:krds00/asym_ensembles.git
```
